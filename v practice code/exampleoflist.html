We promote the basicdesign concepts:

  1. Contrast

    2. Size,color,weight,opacity,etc
    Creates a hierarchy of importance

  2. Repetition

    3. Repeat similar elements (colors,shapes,sizes,etc)to make the design cohesive and balanced

  3. Alignment

    4. All elements should have a locational relationship to each other (lined-up)
    Directs the flow and direction of the user's focus

  4. Proximity

    5. Groups similar components for clearer definitions
    "Blocks" of ideas make a more interesting layout and will more likely hold the users attention
